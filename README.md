# Diabetes Prediction System 🤖🩺

A Machine Learning project that predicts whether a person has diabetes based on health features from the **Pima Indians Diabetes Dataset**. This system includes data exploration, model training, evaluation, and prediction — packaged to showcase ML skills clearly and professionally.

---

## 🧠 Project Overview

In this project, we:
- **Clean and preprocess health data**
- **Train multiple ML models**
- **Evaluate performance using standard metrics**
- **Make predictions on new patient data**

This demonstrates practical machine learning — from data cleaning to model results. :contentReference[oaicite:2]{index=2}

---

## 📊 Features & Models

**Models trained:**
- Logistic Regression  
- k‑Nearest Neighbors  
- Random Forest  
- Decision Tree  

**Key skills shown:**
- Data cleaning (handling missing values, scaling)
- Model selection & comparison
- Evaluation metrics (Accuracy, Precision, Recall, F1)
- Explanation and visualization (plots & insights) :contentReference[oaicite:3]{index=3}

---

## 🗂️ Dataset

We use the **Pima Indians Diabetes Dataset** from Kaggle containing:
- Glucose levels
- BMI
- Blood pressure
- Age
- Insulin, Pedigree function, and more  
- **Target:** Outcome (0 = No Diabetes, 1 = Diabetes) :contentReference[oaicite:4]{index=4}
